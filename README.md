# ATS Resume Analyzer 🚀

An AI-powered Applicant Tracking System (ATS) Resume Analyzer built using Flask and Google Gemini API.  
It analyzes resumes against job descriptions and provides ATS score, keyword match, and improvement suggestions.

## 🔧 Tech Stack
- Python
- Flask
- Google Gemini (GenAI)
- HTML, CSS, JavaScript
- PyPDF2

## ✨ Features
- Upload resume (PDF)
- Paste job description
- ATS score calculation
- Keyword matching
- Resume improvement tips
- Clean and Professional UI
